# cl-aozora

あおぞら文庫フォーマットのパーサ。

## Usage

```lisp
CL-USER> (cl-aozora:parse "もじれつ")
("もじれつ")
CL-USER> (cl-aozora:render ("もじれつ"))
"もじれつ"
```
